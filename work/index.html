Wunderground Weather App

A small project of mine featuring Wunderground API. The app runs with a single ajax request alongside a GeoLocation from the jQuery API. It gathers the data via a JSONP request retriving information such as temprature, wind speed in farhenhite and celcious and giving a visual idea of the weather based on the position retrived via the GeoLocation API.

Jekyll Boilerplate

This was one of my favorite projects and it's used in production for my clients and other developers. Jekyll is a static site generator which is perfect for blogging or portfolios. I built a boilerplate that creates a bare Jekyll site which uses gulp and sass along side gulp-plugins to minify, compile and build Jekyll into the production dir ready to be served. It also uses gulp-connect for a localhost.
